  use the search box

ex: type "ITDC" without the quotes, then once the location is shown, press Enter

  • locations are from a pre-defined list
  • invalid inputs return a world map
  • pressing Enter will include your location using a red marker (if the site can access your location)
  • pressing Enter with an invalid input will show your location
  • a blank input field resets the page to its initial state

for comments, suggestions, corrections, etc. tweet me @josefmonje with the hashtag #upmap please!

this project was created for my HTML/CSS/JS class and uses the following:

  • HTML5 (autofocus, placeholder text, mark, figure & figcaption, geolocation)
  • CSS3 (text shadow, border radius)
  • JQuery (key event handling, setting css properties, iterating over objects, checking matching elements, getting values of elements and attributes)
  • JQuery UI (tooltip)
  • Google Fonts (Droid Sans)
  • Google Static Maps (API v2)
  • Font Awesome (, , )
  • Normalize.css (http://necolas.github.io/normalize.css/)
  • list.js (listjs.com)
  • share.js (sharebutton.co)

Data was derived from the published campus map (the one on the right).

Further improvements can include:

  • mobile-friendly design using CSS media queries
  • separate the data layer by using json format for data and a database for storage
  • allow users to submit suggestions
